# Banburismus Swarm Framework v3.0

O **Banburismus Swarm** é um protocolo de raciocínio multi-agente projetado para eliminar o **Viés de Ancoragem** em LLMs. Inspirado no método de Alan Turing em Bletchley Park, o framework isola o contexto de cada etapa, forçando o sistema a derivar conclusões baseadas apenas na evidência anterior, nunca na intenção original.

## 🧠 Arquitetura de Isolamento

| Agente | Input | Output | Função Crítica |
| :--- | :--- | :--- | :--- |
| **Agente 1** | Problema Original | Premissas [A/B/C] | Identificar incertezas fatais. |
| **Agente 2** | Premissas Classificadas | Pre-mortem | Identificar cenários de falha antes da execução. |
| **Agente 3** | Pre-mortem | Solução Estruturada | Eliminar ambiguidades e adjetivação. |
| **Agente 4** | Solução Estruturada | Crítica Contrafactual | Destruir a solução atual e reconstruir se necessário. |

## 🚀 Como usar

1. Baixe o arquivo `banburismus-swarm.html`.
2. Abra em qualquer navegador moderno.
3. Insira sua OpenAI API Key (armazenada apenas localmente no seu browser).
4. Insira um problema complexo (Ex: "Desenhar um sistema de precificação para SaaS multi-tenant com 99.9% de disponibilidade").
5. Execute e baixe o relatório final em Markdown.
